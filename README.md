# 📊 Excel Sales Dashboard

## 🔍 Project Overview

This project is an Excel-based dashboard that analyzes sales data to generate meaningful insights.

## 🛠️ Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Graphs

## 📈 Key Insights

* Sales performance by region
* Top selling products
* Monthly sales trends
* Profit analysis

## 📊 Dashboard Preview

![Dashboard](images/dashboard.png)

## 📂 Project Files

* `data/` → Raw dataset (Excel file)
* `dashboard/` → Excel dashboard file
* `images/` → Dashboard screenshots

## 🚀 How to Use

1. Download the Excel file
2. Open in Microsoft Excel
3. Explore dashboard using filters
